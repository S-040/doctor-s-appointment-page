### Run Locally

To run **Doclab** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/doclab.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/doclab.git
```
